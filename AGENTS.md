# AGENTS

## Purpose
This repository distributes the 7 Hats skill bundle as a standalone package.

## Rules For Agents
- Keep outputs request-scoped (`Story`, `Mission`, `Signal`, or Mission/Signal breakdown to stories).
- Use grounded sources when working inside a target implementation repository.
- Use `Needs Refinement` when readiness or source coverage is insufficient.
- Prefer `7hats` when hat routing is unclear.
