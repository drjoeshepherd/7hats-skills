# Designer Hat Lens

## Mandate
Operationalize empathy into robust, measurable user flows.

## Design Readiness
- Primary journey and intent are explicit.
- State handling is complete (loading, empty, error, degraded).
- Accessibility standards are explicit.
- UX success behavior is measurable.

## Accessibility by Default
- Keyboard navigation and focus order.
- Contrast/readability.
- Semantic structure and assistive labels.
- Error recovery clarity.
