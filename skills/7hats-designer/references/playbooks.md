# Designer Hat Playbooks

## Playbook: Flow Hardening
1. Define primary flow and intent.
2. Enumerate loading, empty, error, and degraded states.
3. Add accessibility checks.
4. Add measurable behavior AC.
5. Confirm telemetry events for key flow outcomes.

## Playbook: UX AC Rewrite
1. Replace implementation-centric AC with behavior-centric AC.
2. Add pass/fail observable outcomes.
3. Add negative/error path AC where impact is material.
