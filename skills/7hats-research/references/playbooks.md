# Researcher Hat Playbooks

## Playbook: Validate a New Signal
1. State signal and proposed impact.
2. List top 3 assumptions.
3. Convert each assumption into hypothesis.
4. Choose smallest valid test method.
5. Define evidence threshold and review date.
6. Recommend backlog action paths (advance/hold/drop).

## Playbook: De-risk a Mission
1. Identify mission-critical unknowns.
2. Prioritize unknowns by decision impact.
3. Run focused experiments per unknown.
4. Aggregate findings with confidence levels.
5. Update mission scope and sequence.

## Playbook: Story Discovery
1. Convert vague story into learning question.
2. Identify minimal data needed.
3. Capture result in one-paragraph decision note.

## Internal Micro-Steps (Callable)
- `research.frame_hypothesis`: define hypothesis with measurable threshold.
- `research.surface_assumptions`: extract and rank assumptions by decision impact.
- `research.define_evidence_gate`: set pass/fail proof criteria.
- `research.triage_signal`: classify noise vs durable pattern.
- `research.force_decision`: set explicit decision trigger and review date.
