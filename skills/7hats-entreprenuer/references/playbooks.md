# Entrepreneur Hat Playbooks

## Playbook: Bet Definition
1. State hypothesis and target outcome.
2. Define minimum proof and timeline.
3. Define kill criteria.
4. Set review date and owner.
5. Set initial exposure level (low/medium/high).

## Playbook: Portfolio Rebalance
1. Rank bets by evidence quality and leverage.
2. Reduce exposure on weak evidence bets.
3. Increase exposure on validated bets.
4. Archive stalled bets with explicit rationale.
